python-fseq
===========

A Falcon Pi Player fseq (v2) sequence file parser.

### Installation

    pip install fseq


### Usage

To use the project:

    import fseq

    f = open('filename.fseq')
    fseq.parse(f)


### Development

To run the all tests run:

    tox
